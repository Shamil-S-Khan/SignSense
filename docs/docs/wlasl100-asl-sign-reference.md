# WLASL100 ASL Sign Reference

This sheet lists the 100 glosses supported by the current Siformer WLASL100 checkpoint and a direct Signing Savvy lookup page for each word.

## Notes

- The gloss order matches the current checkpoint vocabulary in `frontend/src/lib/sign-recognition/wlasl100.ts`.
- Each link goes to a Signing Savvy search page for that word. Some glosses have multiple ASL variants, so the page may show more than one sign entry.
- Use the search result that matches the intended meaning and movement for your target gloss.
